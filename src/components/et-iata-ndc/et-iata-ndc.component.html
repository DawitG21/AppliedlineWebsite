IATA-New Distribution Capability

Last updated: November 19, 2020

Therefore, IATA-NDC (New Distribution Capability) is a travel industry-supported program (NDC Program) launched by IATA for the development and market adoption of a new, XML-based data transmission standard (NDC Standard). The NDC Standard enhances the capability of communications between airlines and travel agents.

Project Overview

Ethiopian Airlines quest to be the first commercial carrier in Africa to be NDC compatible, hence a partnership with AAAIT was borne in July 2018 to help develop, deploy and transfer knowledge to the Ethiopian Airlines team.

Technology

Developed in C# .Net Core 2.1 with a micro service architecture running on Azure Service Fabric Cluster. However, this cluster has been moved to run on Azure Kubernetes Service (AKS) by the Ethiopian Airlines team.

NDC Scope

Ethiopian Airlines NDC API supports NDC Shopping and partial Order Management at the moment, hence the level 3 certification. There are future plans to include ancillary support, rich media and a full support of Order Management including AirDocExchange, AirDocCancel and AirDocRefund with the current schema 17.2 and more recent schema versions 18.2, 19.2 and 20.2 to meet Level 4 and NDC@Scale certifications.

NDC Certification

AA Advanced Information Technology supported the Ethiopian Airlines team to ensure the API complies with all IATA NDC schema specifications for version 17.2.
